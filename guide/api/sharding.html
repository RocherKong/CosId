# Sharding

分片

# CachedSharding

已缓存的分区，对Sharding子类具体实现的包装，内部通过具体的Sharding进行分区，目的为了通过缓存减少内存对象，提升整体性能。

# IntervalTimeline

按照时间间隔的分区，时间单位根据ChronoUnit, 比如：20220510按照ChronoUnit.MONTHS进行分区，结果分区至202205

# ModCycle

可以提前预知节点数量的场，按照节点取模算法进行分区，比如在4个分区节点：0，1，2，3，分区值为3，4的值会被分区至节点3,0
